3) Reliability/Artifact Flags
- `20240508 meeting_notes.pdf` appears to be slide snapshots on letter pages, with no additional readable speaker-note prose; interpretation therefore relies on figure content, not documented narration.
- Several key lower-panel fields are speckled at fine spatial scales; because one subsequent slide shows strong smoothing/damping, isolated pixel-scale extrema should be treated as low-confidence.
- Unit changes across similar-looking panels (`K`, `W kg^{-1}`, `W m^{-2}`) increase misread risk; comparisons should be made only within a given unit system and colorbar.
- The deck repeatedly reuses near-identical top `\Delta\delta T` panels, so novelty mostly comes from lower-panel definitions/processing; over-interpreting repeated top-panel differences would be an artifact.
- Notes skepticism: OCR on dense equations/symbols is noisy (`ucpaxT`-type artifacts), so all symbolic interpretation above was validated against visual equation structure rather than OCR text alone.
